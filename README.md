# Online Library System

A React-based Online Library System built using React Router and Redux Toolkit.

## Features

* Home Page
* Browse Books Page
* Book Details Page
* Add Book Page
* Redux State Management
* React Router Navigation
* Dynamic Routing
* 404 Not Found Page

## Technologies Used

* React
* React Router DOM
* Redux Toolkit
* React Redux
* Vite

## Installation

```bash
npm install
npm run dev
```
